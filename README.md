# 🚀 Backend Development Practice

This repository contains my hands-on practice while learning backend development using Node.js, Express.js, and MongoDB.

---

## 📌 Project Overview

In this project, I have built a backend server from scratch and implemented core backend functionalities including server creation, database connection, and CRUD operations.

---

## ⚙️ Tech Stack

* Node.js (Core)
* Express.js
* MongoDB
* REST API

---

## 🛠️ Features Implemented

### 🔹 Server Setup

* Created a basic server using Node.js (Core modules)
* Built a server using Express.js for better routing and handling requests

### 🔹 Database Integration

* Connected MongoDB with the server
* Performed database operations using collections

### 🔹 CRUD Operations

* Create new data (POST)
* Read data (GET)
* Update existing data (PUT)
* Delete data (DELETE)

### 🔹 REST API Methods

* Implemented:

  * GET
  * POST
  * PUT
  * DELETE

### 🔹 Data Handling

* Fetched data from MongoDB
* Displayed response in browser using APIs

## ▶️ How to Run the Project

1. Clone the repository
2. Install dependencies:

   ```
   npm install
   ```
3. Start the server:

   ```
   node server.js
   ```
4. Open browser or Postman:

   ```
   http://localhost:3000
   ```
